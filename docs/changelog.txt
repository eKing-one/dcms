定型功能 nick() 
现在你可以在线输出用户名，奖牌和徽章
参数
nick($user = 0, $url = 1, $on = 0, $medal = 0)
1 - ID юзера 
2 - 如果 $url = 1然后输出引用和昵称，如果 0，则仅为昵称
3 - 如果 $on = 1 然后在线输出图标，如果 0，则不输出
4 - 如果 $medal = 1 然后输出奖牌，如果 0，则不输出奖牌
例子：: 
echo user::nick($user['id'], 1, 1, 1); // 返回 <a href="/id1">搜索器</a> [奖章] [在线]
echo user::nick($user['id'], 1, 0, 1); // 返回 <a href="/id1">搜索器</a> [在线]
echo user::nick($user['id'], 1, 1, 0); // 返回 <a href="/id1">搜索器</a> [奖章]
echo user::nick($user['id'], 1, 0, 0); // 返回 <a href="/id1">搜索器</a>
echo user::nick($user['id'], 0, 0, 0); // 将返回搜索器
新增avatar()函数 
显示50px头像和用户图标 
参数
avatar($user = 0, $type = 0)
1-用户ID 
2-我们将输出什么
例子：:
echo user::avatar($user['id'], 0); // 返回 [本身] [图标]
echo user::avatar($user['id'], 1); // 返回 [本身]
echo user::avatar($user['id'], 2); // 返回 [图标]
}
